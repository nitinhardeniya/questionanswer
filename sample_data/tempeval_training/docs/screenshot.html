Annotation of relation types for tasks A and B

Each event from the Event Target List is presented in red bold-faced type. For the first sentence, there are three relations that need to be specified for tasks A and B: one with the time expression today, one with the time expression the night and one woth the document creation time (presented in sentence 0). The interface implements this by displaying three views on the same sentence. Selection of the relation type is by choosing a radio button.
